# Tutorial: Set up the Outlook connector in Azure SRE Agent (preview)

In this tutorial, you connect your SRE agent to Microsoft Outlook so it can send email-based incident notifications and respond to email-triggered workflows.

In this tutorial, you learn how to:

> [!div class="checklist"]
> - Configure OAuth for Outlook access
> - Enable email notifications from your agent
> - Set up email-triggered workflows

## Prerequisites

- An agent created in the SRE Agent portal
- A Microsoft 365 account with Outlook
- Permissions to create Azure AD app registrations
