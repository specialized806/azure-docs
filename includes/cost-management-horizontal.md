Azure services cost money. To help control spending, you can use Microsoft Cost Management to set budgets and configure alerts.

You can analyze, manage, and optimize your Azure costs by using Cost Management. To learn more, see the [quickstart on analyzing your costs](/azure/cost-management-billing/costs/quick-acm-cost-analysis?WT.mc_id=costmanagementcontent_docsacmhorizontal_-inproduct-learn).
